# Submission README

Project: FA550 BTC Futures and Kalshi Contract Behavior Dashboard

## Main Files

- README.md: project overview and run instructions
- requirements.txt: Python dependencies
- app/app.py: Streamlit dashboard
- scripts/: data cleaning, inspection, and weekly build scripts
- data/app/weeks/: weekly dashboard-ready data
- docs/: project documentation, user guide, and screenshots
- ai_logs/: representative AI usage logs

## Run Command

streamlit run .\app\app.py

## Project Focus

This project analyzes BTC futures microstructure and Kalshi BTC binary contract price behavior. It focuses on visualization, threshold distance, decision timing, sensitivity, and futures-side market conditions.
